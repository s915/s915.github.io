Calabi-Yau manifold

2018-09-27 calabi-yau, rgl, visualization

Why this first post ? … When I started this github page, I wanted to add a logo at the top of the main page and I thought of a Calabi-Yau manifold. In my opinion, the 3D representation of that manifold is particularly artistic and this object is used in a theory that I find highly interesting : string theory (Susskind’s website is so cool). Ok the logo is chosen, but I don’t want to take a picture from another website, trim it, etc. I thought “Let’s do it with R”.
With the help of Google, I found this interesting (and apparently well known) paper “A Construction for Computer Visualization of Certain Complex Curves” by Andrew J. Hanson (Indiana University). In his short article, Hanson gives a lot of recipes to visualize complex mathematical objects with a computer. So everything you read in this post isn’t new, it’s taken from Hanson’s article and translated into R.

First we load the packages rgl to dynamically visualize the manifold and colorRamps to add colors to the patches.

library(colorRamps)
library(rgl)

As explained in Hanson’s article, we define three functions $u_i\left(\theta, \xi\right)$, $s(k, n)$ and $z_i(k_i, n, \theta, \xi)$ with $i \in \left\lbrace1,2\right\rbrace$.

The first function $u$ depends on the value of $i$, so there are actually two functions:

$$ u_1\left(\theta, \xi\right) = \dfrac{1}{2} \left( e^{\xi + \imath \theta} + e^{-\xi - \imath \theta} \right) $$ $$ u_2\left(\theta, \xi\right) = \dfrac{1}{2 \imath} \left( e^{\xi + \imath \theta} - e^{-\xi - \imath \theta} \right) $$

u <- function(index, theta, ksi, i = complex(1, 0, 1)) {
  if (!(index %in% 1:2)) {
    stop("index value must be 1 or 2.")
  }

  if (index == 1) {
    0.5 * (exp(ksi + i * theta) + exp(-1 * (ksi + i * theta)))
  } else {
    -0.5 * i * (exp(ksi + i * theta) - exp(-1 * (ksi + i * theta)))
  }
}

The second function $s$ is simply a complex exponential:

$$s(k, n) = e^{\frac{2 \imath \pi k}{n}}$$

s <- function(k, n, i = complex(1, 0, 1)) {
  exp(2 * pi * k * i / n)
}

The third function $z$ is a combination of the previous functions:

$$z_i(k, n, \theta, \xi) = s(k_i,n) \times u_i(\theta, \xi)^{\frac{2}{n}}$$

z <- function(k, n, index, theta, ksi) {
  s(k, n) * (u(index, theta, ksi))^(2 / n)
}

Now we have everything to draw our Calabi-Yau manifold in 3D. We’re going to cover two ranges of values which are $\theta \in \left[0, \dfrac{\pi}{2}\right]$ and $\xi\in\left[-1,1\right]$. To construct a manifold with a particular value of $n$, we define to integers $k_1$ and $k_2$ with $0\leq k_i < n$. Each couple $\left(k_1, k_2\right)$ corresponds to a patch in the manifold that we can colorize with the option color.
Obviously, two complex numbers $z_1$ and $z_2$ give us 4 dimensions. In order to limit our object to 3 dimensions, one of the axis (here $e_3$) has to be a combination of two dimensions. It will be a mix of the imaginary parts of the complex numbers with an angle $\phi = \dfrac{\pi}{4}$. We define:

$$e_1 = \Re\left(z_1(k_1, n, \theta, \xi)\right)$$ $$e_2 = \Re\left(z_2(k_2, n, \theta, \xi)\right)$$ $$e_3 = \cos(\phi)\times\Im\left(z_1(k_1, n, \theta, \xi)\right) + \sin(\phi)\times\Im\left(z_2(k_2, n, \theta, \xi)\right)$$

calabi_yau <- function(
  N, color = FALSE,
  Theta = c(0, pi / 2, 17), Ksi = c(-1, 1, 17), Angle = pi / 4,
  color_default = "white", color_bg = "black"
) {
  k1_ <- seq(0, N - 1)
  k2_ <- seq(0, N - 1)
  ksi_ <- seq(Ksi[1], Ksi[2], length.out = Ksi[3])
  theta_ <- seq(Theta[1], Theta[2], length.out = Theta[3])

  if (color) {
    cols_ <- blue2red(length(k1_)^2)
  } else {
    cols_ <- rep(color_default, length(k1_)^2)
  }

  tryCatch({
    rgl.close()
  }, error = function(e){
  }) 
  
  j <- 1
      
  for (k1 in k1_) {
    for (k2 in k2_) {
      xm <- outer(ksi_, theta_, function(x, t) {
        Re(z(k1, N, 1, t, x))
      })

      ym <- outer(ksi_, theta_, function(x, t) {
        Re(z(k2, N, 2, t, x))
      })

      zm <- outer(ksi_, theta_, function(x, t) {
        cos(Angle) * Im(z(k1, N, 1, t, x)) +
        sin(Angle) * Im(z(k2, N, 2, t, x))
      })
      
      type <- "lines"
      rgl.bg(color = color_bg)
      surface3d(
        x = xm, y = ym, z = zm,
        color = cols_[j], lit = FALSE, front = type, back = type
      )
      
      j <- j + 1
    }
  }
  
  invisible()
}

Let’s have a look at the result for $n=4$ …

calabi_yau(4)

As I mentioned above, by using color = TRUE, you can distinguish the multiple patches created and assembled to form the Calabi-Yau manifold.

calabi_yau(4, color = TRUE)

And just for fun let’s go inside a Calabi-Yau manifold with $n = 12$.

calabi_yau(12, color = TRUE)
